Colourfy

Jquery colour plugin for apply colour on any html elements

Description : Colourfy.js is Aim only for apply font colour This is very light plugin based on jquery It can be work with any version of jquery but i recommand to use v1.5 and above

Author : Freddy Daniel

Version : v1.0.0.0

Requirements : jquery.js

Using this plugin is very simple

If you want colourfy elements Use this

$([selector]).[colour]fy();

Example: below code for apply violet colour for all 'p' elements

$(document).ready(function(){

$("p").violetfy();

})

that all......
